# Puzzle & Dragons Inspired Puzzle

ブラウザ上でパズドラ風の盤面をクリック＆ドラッグで操作できる簡単なパズルです。ドラッグした経路に沿ってドロップが入れ替わり、指を離すと自動でコンボ判定と落ちコン（連鎖）処理が行われます。

## Web版（クリック＆ドラッグ操作）

1. リポジトリのルートで簡易サーバーを立ち上げます。

   ```bash
   python -m http.server 8000 --directory web
   ```

2. ブラウザで <http://localhost:8000> を開きます。（静的ファイルなので `web/index.html` を直接開いても構いません）
3. 盤面のドロップをクリック／タップして好きな方向にドラッグすると、通ったマスと入れ替わって移動します。
4. 指（またはマウスボタン）を離すとコンボを自動で計算し、スコアや消した数、連鎖数が表示されます。
5. 「新しい盤面を作る」ボタンでランダムな盤面にリセットできます。プルダウンから盤面サイズも切り替え可能です。

### 操作のヒント

- ドラッグ中は現在の位置がハイライトされます。斜めには移動できません。
- 盤面を離している間は新しい入力が無効化され、処理が完了すると再び操作可能になります。
- スコアは消したドロップ数に加えてコンボ数と連鎖数に応じたボーナスを計算しています。

## CLI版（テキスト操作）

従来のコマンドライン版も同梱しています。隣接する2マスを指定して入れ替え、コンボを狙います。

```bash
python -m pad
```

主なオプション:

- `--rows`, `--cols`: 盤面サイズ（デフォルトは 5x6）
- `--turns`: プレイするターン数

## テスト

```bash
pytest
```
