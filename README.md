# Puzzle & Dragons Inspired CLI Game

このリポジトリには、人気パズルゲーム「パズル&ドラゴンズ」をモチーフにしたシンプルなコマンドライン版が含まれています。盤面上のドロップを入れ替えて3個以上を揃え、コンボを発生させましょう。

## 遊び方

```bash
python -m pad
```

* `--rows` と `--cols` オプションで盤面サイズを変更できます（デフォルトは5×6）。
* `--turns` で最大ターン数を指定します。
* ターンごとに `row1 col1 row2 col2` 形式（1始まり）で隣接する2マスを指定して入れ替えます。
* `q`, `quit`, `exit` のいずれかを入力するとゲームを終了します。

コンボが発生すると消したドロップ数や連鎖数に応じてスコアが加算されます。ターン終了後には盤面が自動的に補充され、さらに連鎖が起きることもあります。

## テスト

```bash
pytest
```
